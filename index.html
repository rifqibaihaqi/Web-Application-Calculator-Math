Luas Segitiga

Rumus Luas Segitiga yaitu:

L = 1/2 x a x t

Di mana:
L = Luas
a = Panjang Alas
t = Tinggi

Keliling Segitiga

Rumus Keliling Segitiga yaitu:

K = S1 + S2 + S3

Di mana:
K = Keliling
S1 = Sisi a
S2 = Sisi b
S3 = Sisi c

Hitung Keliling Segitiga



Luas Jajar Genjang

Rumus Luas Jajar Genjang yaitu:

L = a x t

Di mana:
L = Luas
a = Alas
t = Tinggi

Hitung Luas Jajar Genjang


Keliling Jajar Genjang

Rumus Keliling Jajar Genjang yaitu:

K = 2 (a + b)

Di mana:
K = Keliling
a = Alas
b = Sisi Miring
